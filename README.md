# SMVP Auto-Office HQ

**The AI-powered business headquarters for MVP Solutions LLC (The Solution MVP)**

Live Dashboard: [smvp-hq.vercel.app](https://smvp-hq.vercel.app)

---

## What Is This?

SMVP HQ is a self-running digital business system built on Next.js, Supabase, and a squad of 6 AI agents. It manages lead generation, sales outreach, product creation, content marketing, operations, and analytics — all from one dashboard you can access on any device.

## Tech Stack

- **Frontend:** Next.js 16 (App Router) + Custom CSS Design System
- - **Database:** Supabase (PostgreSQL + Realtime + RLS)
  - - **Hosting:** Vercel (Hobby plan)
    - - **AI Agents:** 6 prompt-driven agents (Rex, Pixel, Nova, Ace, Flux, Dash)
     
      - ## Agent Squad
     
      - | Agent | Role | What It Does |
      - |-------|------|-------------|
      - | Rex | Research & Prospecting | Finds leads, validates niches, competitor research |
      - | Pixel | Product Designer | Builds websites, Etsy graphics, Canva templates |
      - | Nova | Content & Marketing | Writes listings, emails, scripts, social posts |
      - | Ace | Sales & Outreach | Cold emails, follow-ups, proposals, pipeline management |
      - | Flux | Operations & Delivery | Deploys sites, manages Supabase, pushes code |
      - | Dash | Analytics & Growth | Weekly reports, revenue tracking, growth recommendations |
     
      - ## Revenue Streams (Sequential Unlock)
     
      - Each stream unlocks at $500/mo:
     
      - 1. **Web Design** — Local business websites ($500-$1,500/build)
        2. 2. **Etsy Digital** — Digital products (planners, templates, printables)
           3. 3. **Canva Templates** — Social media kits, pitch decks, resume packs
              4. 4. **Etsy Physical** — Print-on-demand via Printify
                 5. 5. **TikTok Affiliate** — Video content + affiliate links
                   
                    6. ## Dashboard Pages
                   
                    7. - **Dashboard** — Stats overview, agent status, approval queue, revenue chart
                       - - **Agents** — Agent cards with roles, tasks, handoff protocols
                         - - **Pipeline** — Real-time lead tracking and deal pipeline from Supabase
                           - - **Streams** — Revenue stream progress with unlock thresholds
                             - - **Command** — Direct agent commands and quick actions
                               - - **Playbook** — Built-in operations manual and how-to guide
                                
                                 - ## Quick Start
                                
                                 - 1. Clone: git clone https://github.com/TheSolutionMVP/smvp-hq.git
                                   2. 2. Install: cd smvp-hq/dashboard && npm install
                                      3. 3. Copy .env.example to .env.local and add your Supabase keys
                                         4. 4. Run: npm run dev (opens at localhost:3000)
                                           
                                            5. ## Cross-Device Access
                                           
                                            6. The dashboard is hosted on Vercel — open it from any device (PC, laptop, iPad, phone) via the live URL. All data syncs through Supabase in real-time. No installations needed.
